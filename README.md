# CCU-CSIE OOP Mastery: A Modern C++ Textbook

This repository is a structured, academic-grade textbook and exercise set for Walter Savitch’s *Absolute C++* (7th Ed). It is organized into four distinct phases, transitioning from procedural foundations to advanced software architecture.

## Repository Structure

### [Phase 1: Procedural Mastery](./phase_1)
**Focus:** I/O Streams, Formatting, Type Safety, Control Flow, and Chapters 1-5.
- Includes theoretical summaries and 10+ minimalist katas.

### [Phase 2: Object Design](./phase_2)
**Focus:** Encapsulation, Constructors, private/public Access, and Chapters 6-8.
- Includes deep-dives into class design and operator overloading.

### [Phase 3: Memory Management](./phase_3)
**Focus:** Pointers, The Heap, The Big Three (Destructor, Copy Constructor, Copy Assignment), and Chapters 9-10.
- Includes technical guides on preventing memory leaks and managing dynamic resources.

### [Phase 4: Software Architecture](./phase_4)
**Focus:** Separate Compilation, Inheritance, Polymorphism, Virtual Functions, and Chapters 11-16.
- Includes architectural patterns for building scalable C++ systems.

## Standards
- All code is **C++17 compliant**.
- Minimalist, high-signal examples (Katas).
- Professional, technical documentation.
